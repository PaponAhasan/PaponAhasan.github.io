Projects

ATI-S&M

Framework: Flutter Language: Dart

ATI-S&M is a support and management app for the clients with beautiful login UI and astounding live Animated Graphs(Bar, Pie and Line chart)) based on clients’ Ticket priority and Ticket Status for clients’ better UX. Clients can see their ticketing status, close issues, see details, search tickets, rate and send feedback. Clients can submit issues related to their projects as well as send attachments both from camera and gallery. RESTful API was used to connect to back-end services. SQLite was used to store data from rest api. The app is published in Playstore.

Advance App Manager

Framework: Flutter Language: Dart

Advance App Manager is a powerful and easy android application that helps to manage all installed applications and give complete information about the device. The advanced interface helps to find all features easily and monitor hardware in real time.
Dashboard Features: System Information, OS Information, Battery Information, CPU Information, Sensor Information, System Features.
App Features: Sort, Uninstall, Force stop, Backup, Cache clear, Share app, Open in play store, Basic Information, Permission list, Activity list, Provider list, Receiver list, App usage time, Search app. The app is published in Playstore.

The Rising Star

Framework: Flutter; Language: Dart

This is a news app built using Flutter and Dart for both android and ios devices. This app can fetch top headlines from 10 different countries with detailed news from newsapi. There are 7 categories of news for each country. Dark mode is also added for better UX. Other features are beautiful UI, device responsiveness, smooth webview, google firebase notification, live Tv channels and online games to engage user interactions. Apk version of this app can be downloaded from here. Detailed project description is in Github.
